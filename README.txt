Pokémon Unite - Pokédex: React

Pokédex do jogo Pokémon Unite produzida com React no curso PTech, 
com o objetivo de disponibilizar uma lista de Pokémons a qual apresentará 
informações do Pokémon clicado.

Desafios

- React;
- Routes;
- Hooks;
- useState, useEffect;
- API;
- Services.
